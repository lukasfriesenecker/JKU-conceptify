# Conceptify - Concept Mapping Tool

Conceptify ist eine webbasierte Anwendung zur Visualisierung komplexer Ideen, Projekte oder Lerninhalte in Form von interaktiven Concept Maps.

## Tech Stack
* PNPM
* Next.js
* TypeScript
* React
* Interact.js
* Tailwind CSS
* Shadcn/ui
* MongoDB
* Better-Auth
